# Security Login Analysis Insights

## Overview
This dashboard analyzes login activity to identify failed authentication attempts and classify users based on risk level.

## Key Findings
- The highest number of failed login attempts were associated with admin and root accounts, indicating potential targeted access attempts.
- Most users fall under the LOW risk category, suggesting normal system behavior for the majority of activity.
- A smaller subset of users shows repeated failed login attempts, which may indicate brute-force behavior or password guessing attempts.

## Interpretation
The presence of repeated failed login attempts highlights potential security threats that may require monitoring or further investigation.

## Conclusion
This analysis demonstrates how login data can be used to identify risk patterns and support early detection of suspicious activity.
